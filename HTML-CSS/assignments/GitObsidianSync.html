How to setup Git sync SSH with Obsidian

The following must be installed before following

  • git
  • obsidian

Step 1: Generating an SSH key

here is what you can follow if you get lost: Generate a github SSH key

1: Generating the SSH key 

 ssh-keygen -t ed25519 -C "your_email@example.com"
  • Next we are prompted to save the key somewhere, I like to just leave it how it is and press enter.
  • You will then be asked for a passphrase, I also leave this blank but be careful as this is a security concern.

2: Start the ssh-agent in the background 

 eval "$(ssh-agent -s)"
3: Add your SSH private key to the ssh-agent. 
 ssh-add ~/.ssh/id_ed25519
4: Add your public SSH key to yout github account, if you dont know how to do that follow this link: Add SSH key to github account

Step 2: Creating and cloning the repository

1: Make a repository in github and make sure to add a README.MD

2: Now on github copy the repository by SSH

3: In your terminal, CD into a folder where you want to have the repository, I like to use documnets.

4: Next we will git clone the SSH url we copied above

Step 3: Obsidian

1: Open obsidian and open the repository you cloned

2: Install the community plugin GIT and enable it, there are 2 things you need to change/enable.

  • Change the "Auto commit-and-sync interval (minutes)" to 1 minute
  • Enable the pull on startup setting further down
3(optional): If you want to sync all your settings and plugins refer to 3A, if not refer to 3B
3A 3B
Open github or the terminal and make a ".gitignore" file, Go into the file and add ".obsidian/workspace.json" If you dont want to sync all your settings and plugins you can just add the entire .obsidian folder to the .gitignore file

4: Lastly if you see any errors when trying to sync or pull on a diffrent device it may be becasue your .gitconfig isnt set properly setup. Run the commands below on both devices and use your githib credentials.
  • git config --global user.email "your-email@example.com"
  • git config --global user.name "Your Name"
