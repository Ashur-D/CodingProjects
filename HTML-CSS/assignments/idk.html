Setting up Git Sync in Obsidian

Install these before continuing:
Git Obsidian

1 - Generating an SSH key

  1. Generate the SSH key:

    ssh-keygen -t ed25519 -C "your_email@example.com"
    • When prompted for a file location, you can press Enter to accept the default.
    • When asked for a passphrase, it’s safer to set one. If you leave it blank, be aware of the security risk.

  2. Start the ssh-agent in the background:

    eval "$(ssh-agent -s)"

  3. Add your SSH private key to the ssh-agent:

    ssh-add ~/.ssh/id_ed25519
  4. Add your public SSH key to your GitHub account: Add SSH key to GitHub account

2 - Creating and cloning the repository

  1. Create a repository on GitHub and add a README.md.
  2. On GitHub, copy the repository’s SSH URL.
  3. In your terminal, cd into the folder where you want the repository (e.g., Documents).
  4. Clone the SSH URL you copied above.

3 - Obsidian

  1. Open Obsidian and open the repository you cloned.
  2. Install the community plugin “Git” and enable it. Then configure:
    • Set “Auto commit-and-sync interval (minutes)” to 1.
    • Enable “Pull on startup”.
  3. If you want to sync all your settings and plugins, refer to 3A; if not, refer to 3B:
    3A 3B
    Create a .gitignore file (via GitHub or your terminal), and add: .obsidian/workspace.json If you don’t want to sync all settings and plugins, add the entire .obsidian/ folder to .gitignore.
  4. Lastly, if you see errors when syncing or pulling on another device, your .gitconfig may not be properly set up. Run the commands below on both devices with your GitHub credentials: git config --global user.email "your-email@example.com"
    git config --global user.name "Your Name"
